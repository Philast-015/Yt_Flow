# Yt_flow

	Yt_flow is a terminal + web based youtube client that offers videos without any ads and insane customization like:
	- making it minimal and productive
	- gaming related 
	- education only 
	- and much more

## Installation

```bash
git clone https://github.com/philast-015/yt-flow
cd Yt_flow
./install.sh
```

Or through pip

```bash
pip install yt-flow
```
